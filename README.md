# Steam-Omakuni
Steamの地域差を可視化

# Getting Started

## Step 1: Installation

```bash
git clone git@github.com:big-mon/steam-omakuni.git steam-omakuni

cd steam-omakuni

touch .env
echo "MYSQL_ROOT_PASSWORD=[YOUR_PASSWORD]" > .env
echo "TZ='[YOUR_TIMEZONE]'" >> .env
```

## Step 2: Run

```bash
docker-compose up
```
